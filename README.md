## Getting Started

Partytime!


### Installing

Clone from repo,
```
git clone 
```

Install dependencies,
```
npm install
```

Build project,
```
npm run build
```

Then you should be good to go.
```
npm start
```

A dev server will open locally on port 3000. The content on this server can be accessed in any browser by navigating to the URL:
```
http://localhost:3000
```
